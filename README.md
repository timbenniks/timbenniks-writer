# Writer - Rich Text Editor

A beautiful, full-screen rich text editor built with Next.js, React, and TipTap.

## Features

- **Full-screen editing experience** - Distraction-free writing
- **Rich text formatting** - Headings, bold, italic, underline, links, lists, quotes
- **Sleek design** - Modern, clean interface with impeccable UX
- **Light mode** - Optimized for comfortable writing sessions
- **Keyboard shortcuts** - Standard shortcuts for formatting

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the editor.

## Tech Stack

- **Next.js 14** - React framework
- **TipTap** - Rich text editor framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling

## Usage

The editor supports the following formatting options:

- **Headings**: H1, H2, H3
- **Text formatting**: Bold, Italic, Underline
- **Lists**: Bullet lists, Numbered lists
- **Links**: Add and remove links
- **Quotes**: Blockquotes
- **Undo/Redo**: Standard undo/redo functionality
